## Project Description
CineRate is a responsive movie review website developed using
HTML, CSS, and JavaScript. Users can browse movies, view details,
and submit reviews that are stored locally using browser
localStorage.

This project was developed as part of the Introduction to Web
Development course.

## Features
- Multi-page website
- Movie listing page
- Movie review submission
- Ratings system
- LocalStorage data persistence
- Responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## How to Run the Project
1. Download or clone the repository
2. Open `index.html` in a web browser
3. No server or build tools required
