# StepStyle Shoes - E-commerce Website

A fully responsive, multi-page shoe store website built with vanilla HTML, CSS, and JavaScript.

## Project Overview

StepStyle Shoes is a responsive e-commerce website that allows users to browse shoes, add items to a cart, and contact the store. The project demonstrates core frontend web development concepts without using any frameworks or libraries.

## Features

### Pages
1. **Home Page** (`index.html`)
   - Hero section with call-to-action
   - Featured products section
   - Product categories grid
   - Customer testimonials
   - Responsive navigation

2. **Products Page** (`products.html`)
   - Complete product catalog
   - Category filtering
   - Price and name sorting
   - Add to cart functionality
   - Responsive product grid

3. **Contact Page** (`contact.html`)
   - Contact form with validation
   - Store information section
   - Map location placeholder
   - Responsive layout

### Technical Features
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Shopping Cart**: Add/remove items, persists using localStorage
- **Form Validation**: Real-time validation for contact form
- **Accessibility**: Semantic HTML, ARIA labels, keyboard navigation
- **Performance**: Optimized images, lazy loading

## Technologies Used

- **HTML5**: Semantic markup, proper document structure
- **CSS3**: Flexbox, CSS Grid, custom properties (CSS variables), media queries
- **JavaScript (ES6+)**: DOM manipulation, event handling, localStorage API
- **Git**: Version control
- **GitHub**: Code repository and hosting
